Making API Calls

Due to sockets being unavailable in Pyodide, packages such as requests are currently unsupported out of the box.

The package Pyodide-HTTP can be used to patch common http libraries such as requests. See Using Packages for package installation instructions. To use this package, the following Packages object must be used:

{ official: ['pyodide-http'], micropip: ['requests'] }
