# Variables

Une <H>variable</H> permet de stocker une valeur :
```python
a = 3 # définit une variable a qui vaut 3
print(a) # affiche 3
```
Chaque variable possède :
- un <H>nom</H> (ici `a`).
- une <H>valeur</H> (ici `3`), stockée dans la mémoire RAM de l'ordinateur.
- un <H>type</H> (ici `int` pour *integer*, c'est-à-dire un entier).

On peut modifier la valeur d'une variable :
```python
a = 3
a = 42 # la valeur de a est remplacée
print(a) # affiche 42
```
