## Variable locale

Dans la fonction `f` précédente, `x` est une <Highlight>variable locale</Highlight> : elle n'existe que dans le corps de la fonction. On ne peut donc pas y accéder en dehors de la fonction.  
C'est l'analogue d'une variable muette $x$ dans une intégrale $\int_a^b f(x) dx$ en mathématiques.

De même, toute variable définie à l'intérieur d'une fonction est locale à cette fonction :

```python
def f(x):
    y = 3 # variable locale à f
    return x + y

print(f(2))
print(x) # x n'existe pas ici
print(y) # y non plus
```
